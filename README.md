# TaskFlow Pro

## 📌 Project Overview

TaskFlow Pro is a professional and responsive Todo List Web Application developed using HTML, CSS, and JavaScript as part of **Task-2** for Apex Planet Software Pvt Ltd.

The application demonstrates frontend development concepts including DOM manipulation, responsive web design, CRUD operations, form validation, Flexbox, CSS Grid, and LocalStorage integration.

---

# 🚀 Features

## ✅ Todo Management

* Add Tasks
* Delete Tasks
* Move Tasks Between Status Columns
* Task Priority Levels
* Search Tasks Dynamically

## ✅ Image Gallery CRUD

* Add Images Using URLs
* Preview Uploaded Images
* Edit Image Titles
* Delete Images

## ✅ Contact Form Validation

* Name Validation
* Email Format Validation
* Indian Mobile Number Validation
* Subject Validation
* Message Validation

## ✅ Responsive UI/UX

* Flexbox Layout
* CSS Grid Layout
* Mobile Responsive Design
* Smooth Navigation
* Toast Notifications
* Dark Mode Support

## ✅ Data Persistence

* LocalStorage Integration
* Stores Tasks
* Stores Gallery Images
* Stores Contact Form Data
